# Implementación visual

Los archivos de Stitch son HTML estático con Tailwind CDN, Material Symbols e imágenes remotas. La app implementa el mismo lenguaje visual: fondo claro, azul primario, tarjetas redondeadas, navegación inferior móvil, sidebar en escritorio, acciones grandes y formularios en español. Se reemplazó Material Symbols por Lucide para mantener dependencias React consistentes.
